# VisioAlgo
An algorithm visualiser made with the PyGame Library.

Pathfinding controls:
C or R - To clear/reset the screen
ESC - To go back
SPACE - To start the algorithm
Right Click - To remove the blockade or start or finish node
Left Click - Set start node (1st click) \ Set end node (2nd click) \ Set blockade (3rd click and more....)
